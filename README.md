# 🌐 She Can Foundation - Webpage

This project is a responsive and visually appealing webpage developed as part of an **internship evaluation task** for the *She Can Foundation*. It is built using fundamental web technologies like HTML and CSS.


## 📌 Project Description

The purpose of this project is to design a simple, user-friendly webpage that represents the vision and mission of the **She Can Foundation**. The focus is on clean layout, proper structure, and basic styling.



## 🛠️ Technologies Used

- HTML5  
- CSS3  



## 📂 Project Structure

she-can-foundation-task/
│── index.html  
│── style.css  
│── assets/ (images, icons, etc.)



## 🚀 Features

- Responsive design  
- Clean and modern UI  
- Structured layout using semantic HTML  
- Basic styling with CSS
